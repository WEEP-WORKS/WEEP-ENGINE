Weep Engine

What is the engine about?

3D Engine made by Jorge Gemas and Lluís Moreu for the 3D engines subject during the 3rd year of Game Design and Game Development degree at CITM.

The engine was born for the 3d engines assignment and has a User Interface subsytem.

On the top of everything there is the main module called App.cpp . This module manages all the other modules calling in a loop its respective Awake, PreUpdate, Update, Postupdte, Cleanup that they share through a base class Module.h . Do not forget that there is a Main.cpp that has a switch of the different states said before.

Core subsystems

Then the game is divided in 5 main blocks:
  • The Scene where everything is printed managed by a module called SceneManager, including the ImGui menus.


  • The Modules that manage different aspects of the game such as: camera, input, importer, renderer, texture, window, etc…


  • The GameObject structure with its components such as: camera, mesh, texture, transform and the UI components; everything managed by a module called GameObjectManager.


  • The Resources structure that manages the meshes and textures, creating its .meta file just like Unity.


  • The GUI where there are some classes that represent diferent elements of gui such as: images, buttons, checkboxes and input texts.
